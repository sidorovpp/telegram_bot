1) <b>Документы</b> - показывает список документов на согласовании

2) <b>Задачи</b> - показывает непрочитанные задачи

3) <b>Счета</b> - показывает счета без визы

4) <b>Заявки</b> - показывает заявки на продажу квартир для согласования

5) <b>Документ</b> «текст для поиска» - ищет документы по номеру или тексту
Например:
<i>Документ 111111</i>
<i>документ бизнес-план</i>

6) <b>Задача</b> «текст для поиска» - ищет задачи по номеру или тексту
Например:
<i>Задача 123123</i>
<i>задача исправить договор</i>

7) <b>Счёт</b> «текст для поиска» - ищет счёт по номеру или тексту
Например:
<i>Счёт 33333</i>
<i>счет Консультант+</i>

8) <b>Сотрудник</b> «текст для поиска» - ищет сотрудника по части ФИО
Например:
<i>Сотрудник Сидоров</i>
<i>cотрудник Иванов Иван</i>

9) <b>Регламент</b> «текст для поиска» - ищет регламент по описанию
Например:
<i>Регламент заявки</i>
<i>регламент бюджет</i>

10) <b>Клиент</b> «текст для поиска» - ищет клиента по части ФИО
Например:
<i>Клиент Иванов Иван Ивнович</i>
<i>клиент Пётр Петрович</i>

11) <b>Заявка</b> «текст для поиска» - ищет заявку на продажу квартир по номеру или тексту
Например:
<i>Заявка 123123</i>
<i>заявка Петров</i>

12) Можно просто написать лююбой текст для поиска, затем выбрать кнопкой - где искать.
Например:
<i>Сидоров Пётр</i>

13) Чтобы написать в переписку Документа, Счёта, Задачи или Заявки или прикрепить новый файл - ответьте на сообщение, ответ добавится в "Переписку", а файл в "Файлы".
Например, откройте Задачу, выделите сообщение в Телеграм, нажмите "Ответить", напишите ТЕКСТ или прикрепите ФАЙЛ.
