1) <b>Документы</b> - показывает список документов на согласовании

2) <b>Задачи</b> - показывает непрочитанные задачи

3) <b>Счета</b> - показывает счета без визы

4) <b>Документ</b> «текст для поиска» - ищет документы по номеру или тексту
Например:
<i>Документ 111111</i>
<i>документ бизнес-план</i>

5) <b>Задача</b> «текст для поиска» - ищет задачи по номеру или тексту
Например:
<i>Задача 123123</i>
<i>задача исправить договор</i>

6) <b>Счёт</b> «текст для поиска» - ищет счёт по номеру или тексту
Например:
<i>Счёт 33333</i>
<i>счет Консультант+</i>

7) <b>Сотрудник</b> «текст для поиска» - ищет сотрудника по части ФИО
Например:
<i>Сотрудник Сидоров</i>
<i>cотрудник Иванов Иван</i>

8) <b>Регламент</b> «текст для поиска» - ищет регламент по описанию
Например:
<i>Регламент заявки</i>
<i>регламент бюджет</i>

9) <b>Клиент</b> «текст для поиска» - ищет клиента по части ФИО
Например:
<i>Клиент Иванов Иван Ивнович</i>
<i>клиент Пётр Петрович</i>

10) Можно просто написать лююбой текст для поиска, затем выбрать кнопкой - где искать.
Например:
<i>Сидоров Пётр</i>

11) Чтобы написать в переписку Документа, Счёта или Задачи - ответьте на сообщение с текстом, ответ добавится в переписку.
Например, откройте Задачу, выделите сообщение в Телеграм, нажмите "Ответить", напишите ТЕКСТ или прикрепите ФАЙЛ.
